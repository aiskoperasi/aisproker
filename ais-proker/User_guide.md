# PANDUAN PENGGUNA (USER GUIDE) — AIS-PROKER
> **Sistem Monitoring & Evaluasi Program Kerja**

Selamat datang di Panduan Pengguna AIS-Proker. Panduan ini dirancang untuk membantu Anda memahami seluruh fitur dan menu yang tersedia di aplikasi, mulai dari cara login hingga cara melaporkan pencapaian program kerja Anda.

---

## 🔑 1. AKSES & LOGIN

Aplikasi dapat diakses melalui browser Anda (Chrome/Edge/Firefox) dengan langkah berikut:

1.  **Buka Browser**: Masukkan URL aplikasi yang telah diberikan oleh admin (Contoh: `http://ais-proker.test/login`).
2.  **Halaman Login**: Masukkan **Email** dan **Password** Anda.
3.  **Klik Login**: Anda akan diarahkan ke halaman Dashboard utama.
4.  **Logout**: Untuk keluar, klik nama Anda di pojok kanan atas lalu pilih **Logout**.

---

## 🏠 2. DASHBOARD (HALAMAN UTAMA)

Setelah login, Anda akan melihat ringkasan statistik unit Anda:
- **Total Program**: Jumlah seluruh program kerja yang direncanakan.
- **Selesai**: Program yang sudah mencapai 100%.
- **On Progress**: Program yang sedang berjalan.
- **Progress Rata-rata**: Persentase total kesiapan seluruh program kerja unit.

---

## 📅 3. MEMILIH TAHUN AJARAN & UNIT

Data dalam aplikasi ini bersifat dinamis berdasarkan konteks waktu dan departemen.
- **Pilih Tahun Ajaran**: Gunakan dropdown di header untuk memilih tahun yang ingin Anda pantau.
- **Konfirmasi Unit**: Pastikan unit yang tampil di bagian atas sesuai dengan departemen Anda. Data akan terfilter secara otomatis.

---

## 📋 4. MODUL-MODUL APLIKASI

### A. Rencana Strategis (Visi & Misi)
Menu ini berisi landasan dasar unit Anda.
- **Melihat**: Menampilkan Visi dan poin-poin Misi unit.
- **Mengelola**: Anda dapat menambah atau mengedit Visi/Misi jika memiliki hak akses yang sesuai.

### B. Sasaran Mutu (KPI Unit)
Menu ini memonitor target kualitas layanan/departemen.
- **Target**: Nilai persentase yang ingin dicapai (Contoh: 100%).
- **Capaian**: Nilai ini akan **terisi otomatis** berdasarkan rata-rata progres program kerja yang terkait.
- **Metode**: Penjelasan cara pencapaian target tersebut.

### C. Anggaran (Budget)
Menu untuk memantau keuangan unit.
- **Kode**: Kode akun anggaran (COA).
- **Amount**: Pagu anggaran yang disepakati.
- **Realization**: Total dana yang sudah terpakai (diisi secara manual melalui detail program kerja).

### D. Program Kerja (Menu Inti)
Ini adalah menu yang paling sering Anda gunakan untuk melaporkan kegiatan.
1.  **Daftar Program**: Menampilkan seluruh kegiatan yang telah direncanakan.
2.  **Tambah Program**: Klik "Tambah Baru" untuk membuat rencana kegiatan baru (Nama, Deskripsi, PJ, Anggaran).
3.  **Detail Program**: Klik ikon mata untuk masuk ke halaman monitoring detail.

---

## 🚀 5. ALUR MELAPORKAN PROGRES (WORKFLOW)

Untuk menghasilkan laporan yang transparan, ikuti alur berikut:

1.  **Siapkan SOP**: Unggah file SOP (Standar Prosedur) sebelum kegiatan dimulai.
2.  **Buat Indikator**: Di halaman Detail Program, klik **Tambah Indikator**.
    - Pecah kegiatan menjadi beberapa tahap (Contoh: Persiapan, Pelaksanaan, Laporan).
    - Berikan **Bobot** (Contoh: Persiapan 20%, Pelaksanaan 50%, Laporan 30%). Total bobot harus 100%.
3.  **Update Capaian**: Geser slider persentase pada tiap indikator saat Anda menyelesaikan tahap tersebut.
4.  **Unggah Bukti (Evidence)**: Klik ikon kamera/upload pada indikator untuk melampirkan foto/PDF sebagai bukti fisik.
5.  **Realisasi Anggaran**: Input nominal dana yang terpakai pada kolom Realisasi Anggaran.
6.  **Selesai**: Jika akumulasi indikator mencapai 100%, status program akan otomatis berubah menjadi **DONE**.

---

## 📊 6. LAPORAN & EXPORT

Tersedia menu **Laporan** untuk mengunduh rekapitulasi data dalam format Excel:
- **Laporan Unit**: Rekap progres program kerja per unit.
- **Laporan Sasaran Mutu**: Rekap keberhasilan KPI.
- **Laporan Anggaran**: Rekapitulasi penggunaan dana.
